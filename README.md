# Storage Manager Pro - インフラ運用自動化ツール

## 概要
本プロジェクトは、散らかりがちなディレクトリを拡張子に基づいて自動整理する、Windows向けデスクトップアプリケーションです。
単なるファイル移動に留まらず、インフラエンジニアの実務で重要視される**「データの整合性保証」**と**「運用の透明性」**を兼ね備えた設計を行っています。

## 主な機能
1. **インテリジェント・ファイル整理**:
   - 拡張子ごとに指定されたフォルダへ自動仕分け。
   - 実行日時に基づく世代管理フォルダの自動生成。
2. **整合性検証 (SHA-256)**:
   - ファイル移動前後のハッシュ値を比較し、1ビットの欠損も許さない確実なデータ移行を保証。
3. **監査ログの自動生成**:
   - `logs/` フォルダ内にタイムスタンプ付きの操作ログを出力。トラブルシューティングを容易にします。
4. **GUIによる設定の永続化**:
   - アプリ画面から整理ルールを追加・更新可能。設定は `settings.json` に動的に保存されます。
5. **ポータブルな実行形式 (EXE)**:
   - Python環境が構築されていない端末でも、ダブルクリックのみで即座に利用可能。

## 技術スタック
- **Language**: Python 3.14
- **Library**: `tkinter` (GUI), `hashlib` (Security), `shutil` (File Ops)
- **Packaging**: PyInstaller
